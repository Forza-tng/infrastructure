Haiku Repository

A hpkg repository continiously providing updates to Haiku's core operating system files.

pkgman add-repo \
https://eu.hpkg.haiku-os.org/haiku-repo/$(getarch)/current

Haikuports Repository

A hpkg repository continiously providing fresh HaikuPorts packages.

pkgman add-repo \
https://eu.hpkg.haiku-os.org/haikuports-repo/$(getarch)/current
